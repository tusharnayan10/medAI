# Understanding Users’ Security, Privacy, and Trust Concerns in Healthcare AI Systems

This repository accompanies our study on users’ Security, Privacy, and Trust (SP&T) concerns regarding healthcare AI systems. The repository provides the resources used to support our longitudinal analysis and qualitative coding process.

## Repository Contents

### 1. Event Timeline
The `events.csv` file contains the complete timeline of major healthcare AI–related events used in our longitudinal analysis.

The event set includes:

- Foundation model and healthcare AI model releases
- Healthcare AI deployments and commercialization milestones
- Regulatory and governance developments
- Security, privacy, and trust incidents
- Clinical safety and reliability events
- Human oversight and accountability developments

For each event, we provide:

- Event Name
- Event Date
- Event Type

The timeline was compiled through manual review of official announcements, regulatory documents, industry reports, and primary sources. Rather than constructing an exhaustive historical record, we focused on events likely to influence public discussions and perceptions of healthcare AI risks.

---

### 2. SP&T Codebook
The repository also contains the qualitative codebook used during our thematic analysis.

The codebook includes:

- Theme definitions
- Sub-theme definitions
- Inclusion and exclusion criteria
- Example excerpts
- Annotation guidelines

The codebook was developed through an iterative qualitative coding process and was subsequently used to support large-scale annotation of healthcare AI discussions.
